Aide de mémoir des sélector css deja vue

  • * selectionne tous les élement de la page html
  • .ma-jolie-navsélectionne tous les élement dans le nom de class "majolie nav"
  • #mon-unique-idsélectionne l'element ayant le nom d'id "mon-unique-id"
  • div,articlesélectionne tous les balises < div> , <nav>
  • nav a sélectionne les liens <a> inclu dans les balises <nav>
  • nav a :hover sélectionne tous les lien de la nav et active une animation lors de la survol de souris
